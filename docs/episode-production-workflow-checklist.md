# Episode Production Workflow Checklist

## Goal

This checklist defines the repeatable workflow for taking a Claro English episode from approved script to published content package.

## Use Case

Use this for Episode 1 now, then adapt it into the standard operating process for all future episodes.

## Phase 1: Content Lock

### 1. Strategy and outline

- Confirm episode topic
- Confirm learner level
- Confirm practical situation focus
- Approve episode outline

### 2. Script creation

- Write clean episode script
- Review realism and natural phrasing
- Finalize key teaching phrases
- Confirm next-episode bridge

### 3. Production script prep

- Convert to production-ready script
- Add pacing cues
- Add pause timing
- Add visual cues
- Add subtitle highlight list

## Phase 2: Voice and Audio Production

### 4. Voice prep

- Confirm Emma voice settings
- Confirm Daniel voice settings
- Confirm officer voice settings
- Review pronunciation watchlist
- Review signature lines

### 5. Voice generation or recording

- Generate or record Emma lines
- Generate or record Daniel lines
- Generate or record officer lines
- Export clean takes

### 6. Audio review

- Review pacing and clarity
- Confirm repeat-after-me pauses are long enough
- Fix mispronunciations
- Balance speaker tone and volume
- Export final master audio

## Phase 3: Visual Production

### 7. Visual asset prep

- Finalize host visuals
- Finalize logo and brand elements
- Prepare section cards
- Prepare phrase cards
- Prepare quote cards
- Prepare tip cards
- Gather airport b-roll or illustrations
- Create thumbnail assets

### 8. Video edit assembly

- Place intro card
- Sync host audio
- Add scene cards
- Add phrase overlays
- Add b-roll and host visuals
- Add practice pauses visually
- Add end card and CTA

### 9. Subtitles and readability

- Add subtitles for full episode
- Highlight key phrases where needed
- Check subtitle timing for beginners
- Confirm mobile readability

## Phase 4: Episode Packaging

### 10. Episode metadata

- Finalize YouTube title
- Finalize YouTube description
- Finalize podcast episode summary
- Finalize website meta description
- Finalize thumbnail text

### 11. Supporting assets

- Create show notes
- Create Instagram caption
- Create Instagram carousel copy
- Create X post or thread
- Prepare episode page copy

## Phase 5: QA and Publish

### 12. Quality assurance

- Watch full video once with sound
- Watch again with subtitles only
- Check phrase cards for text accuracy
- Check host visuals for consistency
- Confirm end card and CTA are correct
- Confirm no pacing feels rushed

### 13. Publish package

- Export final YouTube video
- Export podcast audio version
- Upload YouTube episode
- Upload podcast audio to feed host
- Publish or stage website episode page
- Schedule social posts

### 14. Archive and systemize

- Save final script and production files
- Save thumbnail and export assets
- Save final metadata copy
- Record lessons learned for future episodes
- Reuse templates for Episode 2

## Episode 1 Recommended Working Order

For this first episode, use this exact sequence:

1. Lock the production script
2. Lock the shot list and storyboard
3. Lock the visual asset checklist
4. Generate or record voice tracks
5. Build cards, phrases, and thumbnail
6. Edit the full video
7. Add subtitles
8. Review the final cut
9. Publish with the supporting assets

## Team Roles By Function

If one person handles everything, use this as a task sequence.
If multiple people are involved, split like this:

- Creative lead: script, final messaging, host consistency
- Voice production: Emma, Daniel, officer tracks
- Editor: video assembly, subtitle timing, exports
- Publisher: YouTube, podcast, website, social packaging

## Ready-to-Publish Checklist

Before Episode 1 goes live, confirm:

- Final video export is approved
- Audio-only export is approved
- Thumbnail is approved
- YouTube title and description are approved
- Podcast summary is approved
- Social captions are ready
- Website copy is ready
- All files are saved in the episode folder

## Future Automation Potential

Once this workflow is proven manually, parts of it can be automated:

- Script packaging
- Metadata generation
- Show notes generation
- Subtitle draft generation
- Website episode page generation
- Social draft generation

Human review should still remain for:

- Final voice quality
- Final video edit
- Thumbnail approval
- Public publishing

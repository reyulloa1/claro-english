# Episode 1 Visual Asset Checklist

## Goal

This checklist covers the visual assets needed to produce Episode 1 for YouTube and related publishing surfaces.

## Episode

- Series: Claro English
- Episode title: First Day in the U.S.: At the Airport

## Core Branding Assets

### Required

- Claro English logo
- Primary tagline lockup: Real English for Real Life in the U.S.
- Brand color palette
- Brand typography choices
- Lower-third style for host names
- End card template

### Recommended

- Secondary supporting tagline lockups
- Social-safe square and vertical crop variants
- Reusable section card template
- Reusable phrase card template

## Host Assets

### Emma

Required:
- Primary host portrait or character render
- Transparent PNG version if needed
- Close-up crop for thumbnails or phrase sections
- Branded host lower-third

### Daniel

Required:
- Primary co-host portrait or character render
- Transparent PNG version if needed
- Close-up crop for thumbnails or phrase sections
- Branded co-host lower-third

### Officer

Optional:
- Minimal visual treatment only
- No full branded character needed unless desired

Recommendation:
Use a simpler officer visual or silhouette so Emma and Daniel remain the focus.

## Episode-Specific Visual Assets

### Intro and Outro

Required:
- Intro title card
- Outro / end card
- Subscribe CTA card

Text needed:
- Claro English
- Real English for Real Life in the U.S.
- Subscribe for the next lesson

## Background Visuals / B-Roll

Required:
- Airplane landing visual
- Airport terminal walking visual
- Passport control line visual
- Close-up travel document or passport visual
- General airport waiting visual

Optional:
- Airport signage visual
- Suitcase wheel / baggage visual
- Arrival board visual

Recommendation:
Keep b-roll realistic but calm. Avoid crowded, chaotic, or stressful-looking imagery.

## Icon Set

Recommended icons:
- Passport
- Airplane
- Suitcase
- Clock or calendar
- Hotel
- Home
- Speech bubble
- Question mark

## Thumbnail Assets

Required:
- Emma and Daniel thumbnail cutouts or portraits
- Episode title text
- Background airport visual
- Claro English logo mark

Suggested thumbnail text options:
- English at the Airport
- First Day in the U.S.
- Airport English for Beginners

Thumbnail composition recommendation:
- Emma and Daniel on one side
- Bold title on the other side
- Airport background lightly blurred

## Subtitle / Caption Assets

Required:
- Subtitle style template
- Highlight color for key phrases
- Safe subtitle position for YouTube and mobile

Recommendation:
Keep subtitles large and easy to read. Avoid placing them too low on screen where interface controls may overlap.

## Export Variants

Recommended exports:
- 16:9 YouTube main video
- 9:16 short clips from best moments later
- 1:1 or 4:5 crops for Instagram graphics if reused

## Asset Folder Recommendation

Suggested folder structure:

- assets/branding/
- assets/hosts/emma/
- assets/hosts/daniel/
- assets/episode-01/cards/
- assets/episode-01/phrases/
- assets/episode-01/broll/
- assets/episode-01/thumbnail/
- assets/episode-01/exports/

## Completion Checklist

Before editing begins, confirm:

- Emma and Daniel visuals are finalized
- Section card template exists
- Phrase card template exists
- All required phrase text is prepared
- Thumbnail concept is chosen
- B-roll source set is ready
- Subtitle style is locked
